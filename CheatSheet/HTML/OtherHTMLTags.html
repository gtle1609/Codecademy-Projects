Semantic Tags

Tags Names Description
<header> Header describes the content at the top of the page, may include a logo, navigational links or a search bar
<nav> Navigation encapsulated the page's navigational links, often placed inside the <header> or <footer>
<main> Main encapsulates the main content of a page between the header/navigation and the footer areas
<footer> Footer includes the page's footer content
<figure> Figure encapsulate media such as an image, diagram or code snippet
<figcaption> Figcaption describe the media encapsulated within <figure>, normally use within <figure> to group media and description
<section> Section defines elements in a document, such as chapters, headings, or any other area of the document with the same theme
<article> Article holds content that makes sense on its own such as articles, blogs, and comments
<aside> Aside mark additional information that can enhance another element but isn't required in order to understand the main content, usually in a sidebar or a location where it doesn't obstruct the main piece of content

Nonsemantic Tags

Tags Names Description
<li> List Item create list items inside <ol> or <ul>
<ol> Ordered List create a list of items in sequential order, each list item appears numbered by default
<ul> Unordered List create a list of items in no particular order, each individual list item will have a bullet point by default
<em> Emphasis emphasizes text in italics
<strong> Strong highlight important, serious, or urgent text and browsers will render highlighted text in bold
<span> Span inline container for text and can be used to group text for styling purpose
<html> HTML root of an HTML document, should be added after the !DOCTYPE declaration
<head> Head metadata, contains general information about an HTML page, title and links to stylesheets, that isn't displayed on the page,
<title> Title contains a text the defines the title of an HTML document, contained inside <head> element
<h1>-<h6> Heading 6 levels of heading elements, ordered from the highest <h1> to the lowest level <h6>
<body> Body represents the content of an HTML document, content inside are rendered on the web browsers
<p> Paragraph contains and displays a block of text
<div> Division used as a container that divides an HTML document into sections, can contain flow content such as headings, paragraphs, links, images, etc.
<br> Line Break create a line break in text, useful where a division of text is required, requires only an opening tag
<a> Anchor create hyperlink
<video> Video allows to add videos to website, requires a closing tag
<audio> Audio allows to implement audio into website, requires a closing tag
<embed> Embed self-closing tag, can be used to implement any type of media
<img> Image self-closing tag, embed images in documents

Attributes

Attribute Names Description
name="value" Name and Values configure or change the behavior of the element
id="" ID differentiate among the elements
alt="" Alternative Text displayed if an image fails to render due to an incorrect URL, image format is not supported by the browser, image is blocked from being displayed, or image has not been received from the URL
href="" Hyperlink Reference determines the location the anchor element points to
target="" Target specifies where a hyperlink should be opened
