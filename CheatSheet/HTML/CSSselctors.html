CSS Selectors

Selectors Description
Type match all elements of a given type or tag name
Class match elements based on the contents of their class attribute, reusable and apply to many elements
ID match elements based on the content of their id attribute, unique and style only a single element
Groups of Selectors match multiple selectors to the same CSS rule, using a comma-separated list
Chaining Selectors apply only to elements that match all criteria
CSS Descendant Selector match elements that are descended from another matched selector

CSS Specificity

(Most specfic) ID selectors > Class selectors > Type selectors (Least specific)

Used when there are multiple conflicting property values that point to the same element
