Table Tags

Tag Name Description
<table> Table Creates a table and wrap all the content for HTML table
<tr> Table Row Add rows to a table before adding table data and table headings
<td> Table Data Add a cell of data to a table, can be nested inside <tr>
<th> Table Heading Add titles to rows and columns of a table and must be nested inside <tr>
<thead> Table's Head Semantic element which defines the headings of a table columns encapsulated in table rows
<tbody> Table's Body Semantic element which contains all table data other than table heading and table footer content
<tfoot> Table's Semantic element which uses table rows to give footer content or to summarize content at the end of a table

Table Attributes

Attribute Value Name Description
rowspan Set to 1 by default Any positive integer up to 65534 Row Span Indicates how many rows that particular cell should span within the table
columnspan Any positive integer up to 1000 Column Span Indicates how many columns that particular cell should span within the table
