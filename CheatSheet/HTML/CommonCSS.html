Common CSS Properties

Properties Description
font-size set text sizes, values can be many different units
background-color controls the background color of elements
opacity control the transparency of an element, value ranges from 0 (transparent) to 1 (opaque)
font-weight set the weight (boldness) of text
text-align set the text alignment of inline contents
color set foreground text color of an element
background-image set the background image of an element
font-family specify the typeface in a rule set
